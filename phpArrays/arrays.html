    PHP Arrays

    Arrays in PHP

    In PHP, an array is a container that holds multiple values, allowing you to store and manage data efficiently. Arrays in PHP can store any type of data, such as integers, strings, or even other arrays. There are three types of arrays in PHP: indexed arrays, associative arrays, and multidimensional arrays.


    8.1 Indexed Arrays

    An indexed array uses numeric indices to access its elements. By default, PHP assigns numeric indices starting from 0, but you can also define your own indices if needed.


    • Syntax Example:
      • $fruits = array("Apple", "Banana", "Cherry");
      echo $fruits[1]; // Outputs: Banana
    • Explanation:

      • This example demonstrates how to access elements in an indexed array. The array $fruits stores three elements, and you can access any element by referencing its index.

    8.2 Associative Arrays

    An associative array uses named keys instead of numeric indices. This allows you to associate values with descriptive keys, making the array easier to understand and use.


    • Syntax Example:
      • $person = array(
      "name" => "John Doe",
      "age" => 30,
      "email" => "johndoe@example.com"
      );
      echo $person["name"]; // Outputs: John Doe
    • Explanation:

      • This example demonstrates an associative array where each key is a meaningful descriptor (e.g., "name", "age", "email"). You can access the corresponding values using these keys.

    8.3 Multidimensional Arrays

    A multidimensional array is an array that contains other arrays as its elements. This type of array is useful for storing complex data structures such as tables or matrices.


    • Syntax Example:
      • $students = array(
      array("John", "A", 95),
      array("Jane", "B", 88),
      array("Paul", "A", 92)
      );
      echo $students[1][0]; // Outputs: Jane
    • Explanation:

      • This example shows a two-dimensional array. Each element is an array containing multiple values. The second student's name is accessed using [1][0].

    8.4 Conclusion

    Arrays are a powerful feature of PHP that help you store, organize, and manage data efficiently. By understanding the different types of arrays—indexed, associative, and multidimensional—you can write more flexible and dynamic PHP scripts.
