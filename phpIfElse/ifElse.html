    PHP If,If-else

    Conditional Statements

    PHP provides conditional statements such as if, if-else, and else-if to control the flow of execution based on certain conditions.


    4.1 if-else Statement

    The if-else statement allows you to execute one block of code if a condition is true, and another block if the condition is false. It is used when you need to make decisions and control the flow of your program based on boolean expressions.


    • Syntax Example:
      • $age = 20;
      if ($age >= 18) {
      echo "You are an adult.";
      } else {
      echo "You are not an adult.";
      }
    • Explanation:

      • In this example, if the condition ($age >= 18) is true, the first block of code will execute, and the message "You are an adult." will be displayed. If the condition is false, the second block of code will execute, showing "You are not an adult."

    4.2 Useful Functions:
    • if — The basic conditional statement to execute code if a condition is true.
    • else — Provides an alternative block of code if the condition is false.
    • elseif — Allows for multiple conditions to be checked in sequence.
