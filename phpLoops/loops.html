    PHP Loops

    6. Loops in PHP

    In PHP, loops are used to execute a block of code repeatedly, as long as a condition is true. Loops are very useful when you need to perform repetitive tasks like iterating through arrays or performing calculations.


    6.1 for loop

    The for loop is used when you know the number of iterations in advance. It consists of three parts: initialization, condition, and increment/decrement.


    • Syntax Example:
      • for ($i = 1; $i <= 5; $i++) {
      echo "Number $i <br>";
      }
    • Explanation:

      • In this example, the for loop will run 5 times, outputting the numbers 1 to 5, each on a new line.

    6.2 while loop

    The while loop executes a block of code as long as the condition is true. The condition is checked before each iteration.


    • Syntax Example:
      • $i = 1;
      while ($i <= 5) {
      echo "Number $i <br>";
      $i++;
      }
    • Explanation:

      • This example will also output the numbers 1 through 5, as long as the condition $i <= 5 is true.

    6.3 do-while loop

    The do-while loop is similar to the while loop, but it guarantees that the code will execute at least once, even if the condition is false initially.


    • Syntax Example:
      • $i = 1;
      do {
      echo "Number $i <br>";
      $i++;
      } while ($i <= 5);
    • Explanation:

      • This loop also outputs the numbers 1 to 5, and it will execute the code at least once, even if $i starts out greater than 5.

    6.4 foreach loop

    The foreach loop is specifically used for iterating over arrays. It is the easiest way to loop through each element of an array.


    • Syntax Example:
      • $colors = array("Red", "Green", "Blue");
      foreach ($colors as $color) {
      echo "$color <br>";
      }
    • Explanation:

      • This loop will output each color in the array one by one, making it easy to work with arrays.

    6.5 Conclusion

    PHP provides different types of loops for various scenarios, and knowing when to use each one is essential for controlling the flow of your program. Whether you're working with arrays or repeating tasks, loops in PHP are a powerful tool for writing clean and efficient code.
