    PHP Switch

    Switch Statement

    The switch statement in PHP is used to handle multiple conditions based on the value of a single expression. It is often more efficient and easier to read than using multiple if-else statements when you need to check for many different possible values of the same variable.


    5.1 Syntax

    The switch statement evaluates an expression and compares it with different possible values (called cases). If a match is found, the corresponding block of code is executed. Optionally, a default block can be added to execute code if no case matches.


    • Syntax Example:
      • $day = 3;
      switch ($day) {
      case 1:
      echo "Monday";
      break;
      case 2:
      echo "Tuesday";
      break;
      case 3:
      echo "Wednesday";
      break;
      default:
      echo "Invalid day";
      }
    • Explanation:

      • In this example, the value of the $day variable is compared to each case. Since $day is 3, the code block under case 3: will execute, and the output will be "Wednesday". The break statement is crucial as it stops the execution from continuing to the next case once a match is found. If no cases match, the default block will execute (if provided).

    5.2 Useful Notes:
    • The break statement is used to exit the switch statement after a matching case is found. Without it, PHP will continue to evaluate the subsequent cases.
    • The default case is optional but can be helpful for handling unexpected values.
