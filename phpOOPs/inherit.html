    PHP Inheritance

    10. PHP Inheritance

    Inheritance is one of the fundamental concepts of Object-Oriented Programming (OOP). Inheritance allows a class to inherit the properties and methods of another class. This helps in reusing code and creating a hierarchy of classes. In PHP, inheritance is implemented by using the extends keyword.


    10.1 Basic Inheritance

    In PHP, a class can inherit from another class, which allows the child class to use the methods and properties of the parent class. This is done using the extends keyword when defining the child class.

    • Syntax Example:
      • class Animal {
           public $name;
           public function __construct($name) {
               $this->name = $name;
          }

           public function speak() {
               echo $this->name . " makes a sound!";
          }
      }

      // Child class that inherits from Animal
      class Dog extends Animal {
           public function speak() {
               echo $this->name . " barks!";
          }
      }

      $dog = new Dog("Rex");
      $dog->speak(); // Output: Rex barks!

      In this example, the Dog class inherits from the Animal class. The Dog class overrides the speak() method of the parent class to provide its own implementation. When the speak() method is called on the Dog object, it outputs "Rex barks!"

    10.2 Constructor Inheritance

    If a parent class has a constructor, it can be called in the child class using the parent::__construct() method. This is useful for ensuring that the parent class is properly initialized before any additional logic in the child class is executed.

    • Syntax Example:
      • class Animal {
           public $name;
           public function __construct($name) {
               $this->name = $name;
          }

      }

      class Dog extends Animal {
           public $breed;

           public function __construct($name, $breed) {
               parent::__construct($name);
               $this->breed = $breed;
          }

      }

      $dog = new Dog("Rex", "Labrador");
      echo $dog->name; // Output: Rex
      echo $dog->breed; // Output: Labrador

      In this example, the Dog class calls the parent class's constructor using parent::__construct($name) to initialize the name property, and then adds its own property breed.

    10.3 Overriding Methods

    In PHP, a child class can override methods defined in the parent class. This allows the child class to modify the behavior of the inherited methods. The method in the child class must have the same name and parameters as the one in the parent class.

    • Syntax Example:
      • class Animal {
           public function speak() {
               echo "Animal makes a sound!";
          }
      }

      class Dog extends Animal {
           public function speak() {
               echo "Dog barks!";
          }
      }

      $dog = new Dog();
      $dog->speak(); // Output: Dog barks!

      In this example, the speak() method in the child class Dog overrides the method in the parent class Animal. When the speak() method is called on a Dog object, it outputs "Dog barks!" instead of the original message from the parent class.

    10.4 Key Points about Inheritance in PHP
    • Code Reusability: Inheritance allows child classes to reuse code from the parent class, reducing duplication and improving maintainability.
    • Access to Parent Class Methods and Properties: The child class inherits all non-private methods and properties of the parent class.
    • Method Overriding: Child classes can override parent class methods to change their behavior.
    • Single Inheritance: PHP supports single inheritance, meaning a class can inherit from only one parent class.
    • Visibility: Inherited properties and methods are subject to visibility rules. A private method or property from the parent class will not be accessible in the child class.
